# potential-eureka


## Description

A Note Taker application that can be used to write and save notes.

This application uses express.js back end and will save and retrieve note data from a JSON file

Front end was supplied, only backend was developed.

Deployed through Heroku.


github link: https://github.com/Erfuric/potential-eureka

## User Story

AS A small business owner
I WANT to be able to write and save notes
SO THAT I can organize my thoughts and keep track of tasks I need to complete
